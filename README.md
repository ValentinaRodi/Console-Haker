# Console-Haker

Создан компонент с анимированной кнопкой, очищающей консоль (кнопка неактивна, если консоль пуста) и кнопкой, меняющей цвет шрифта

#Cтэк:

• React

• React GlobalStyle

• Function Components React

• Node.js
